## Base URL

All API endpoints are prefixed with `/api/admin`.

## API Overview

The AB Academy Admin API provides a comprehensive set of endpoints for managing all aspects of the educational platform, including:

- User management (students, teachers, admins)
- Group and class management
- Course management
- Exam scheduling and grading
- Event scheduling
- Dashboard analytics
- Chat functionality
- Archive management

Each section of the documentation includes detailed information about the available endpoints, request/response formats, and example usage.
